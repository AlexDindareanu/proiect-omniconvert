# Tutoring Session Scheduling System

This project is a tutoring session scheduling system. The tutor sets available time slots, students can see free slots and book one. Sessions are 2 hours long, groups of 2-4 students.

## Stack

- **Frontend**: React + TypeScript
- **Backend**: NestJS + TypeScript
- **Database**: PostgreSQL
- **Message Queue**: RabbitMQ
- **Infrastructure**: Docker Compose
